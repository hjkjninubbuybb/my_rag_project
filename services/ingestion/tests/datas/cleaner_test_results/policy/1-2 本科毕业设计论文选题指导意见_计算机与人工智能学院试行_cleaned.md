# 计算机与人工智能学院本科毕业设计选题类型说明

本科毕业设计选题分为 论文类型 与 设计类型 两大类，两类选题均需坚守正确政治方向，符合专业培养目标，确保具备专业性、综合性与实践性，助力实现对学生的系统性能力训练。

# 一、论文类型选题

论文类型选题以理论研究与算法探索为核心，聚焦计算机与人工智能领域的学术性、技术性问题，严格遵循“一人一题、独立完成”原则，严禁综述类课题，杜绝题目过于宽泛或狭窄。选题需紧密结合指导教师科研背景、在研项目，或对接学科前沿理论与行业技术瓶颈，或基于企业/科研机构的“真题”，体现一定的学术深度与创新性，符合专业培养对理论分析能力的要求。

该类型侧重引导学生开展算法设计、模型构建、理论推导、实验验证及结果分析等工作，要求学生整合专业基础理论与学术研究方法，完成从文献调研、课题论证、方案设计到实验实施、结论凝练的全流程训练。选题需服务于学科发展或行业技术升级需求，可跨学科融合，鼓励与科研院所联合拟定，通过系统性研究切实提升学生的理论思辨、学术写作及科研创新能力，最终形成兼具学术性与参考价值的毕业论文。

样例：1 基于扩散模型的多尺度特征增强红外可见光图像融合方法研究2 基于TextRank 的期货研报自动摘要方法研究

# 二、设计类型选题

设计类型选题以工程实践与系统实现为核心，紧扣生产、科研、管理等实际需求，突出技术应用性与综合性，优先鼓励以产业需求为导向，与企事业单位联合拟定真实项目类题目，实现“真题真做”。选题需明确技术路线与实现目标，避免空泛化，若涉及多人协作课题，必须清晰界定每名学生独立完成的任务模块，严格控制团队合作选题数量。

该类型侧重引导学生完成完整系统的设计、开发、测试与优化工作，涵盖需求分析、架构设计、代码编写、功能调试、性能评估等全流程工程实践。选题需体现专业技术的综合应用，适配行业发展需要，可结合大创计划、学科竞赛等自主拟定（经指导教师审核批准），着重培养学生的工程思维、技术落地及问题解决能力。最终成果以系统实体（或可运行原型）形式呈现，确保符合行业技术规范与学院质量要求。

两类选题均需经过指导教师确认、系/教研室审核小组审查、学院专家审定的三级审核流程。通过差异化选题设计，兼顾理论研究与工程实践，企业/科研机构的“真题”与导师在研横向、纵向课题，实现因材施教，全面提升学生综合素养与就业、深造竞争力。

样例：1 基于Spring Cloud 的公寓工单系统的设计与实现2 基于微信小程序的电子产品商城系统的设计与实现

# 计算机与人工智能学院本科毕业设计选题指导意见

毕业设计（论文）是本科人才培养的重要教学环节，是实现教学、科研与社会实践相结合的关键点，对提高学生的综合能力和全面素质具有重要意义。为了进一步规范本科毕业设计（论文）工作，确保本科毕业设计（论文）质量，提高我院毕业论文（设计）质量，现就选题工作提出如下要求。

毕业论文（设计）的选题要坚定政治方向、价值取向，选题必须符合《中央关于意识形态工作的重大部署和基本要求》的精神和要求，以“加强社会主义核心价值体系建设，积极培育和践行社会主义核心价值观”为主旨，弘扬正能量。【政治站位】

# 一、指导教师命题

1.指导教师应结合自身的科研背景和在研科研项目提前准备毕业设计题目，确保题目数量充足，原则上每位学生应有1-2个备选题目。同时，应避免题目过于宽泛或狭窄。 综述类课题和涉及国家机密的课题不能作为本科毕业设计（论文）课题 。如涉及多人共同完成的课题，需明确每名学生独立完成的任务。【数量和质量】2.指导教师准备的课题应紧密围绕专业培养目标和毕业要求，体现教学与科学研究、技术开发、经济建设和社会发展紧密结合的原则。同时，充分结合实践，突出创新，能够适应生产或行业发展需要，能够更好地服务国家、区域经济建设和发展。鼓励以产业需求为导向，与科研院所、企事业单位等联合拟定毕业设计题目；鼓励不同专业或不同学科之间的相互结合和交叉。【选题来源要求】3.题目应体现一定的专业性、学术性、技术性、综合性和创新性，应有利于各类学生水平和能力的发挥与提高，体现因材施教，同时，能够体现对学生能力的系统性和综合性训练。
# 二、学生选题

学生应优先在指导老师给定的课题中选择毕业设计题目，也可结合自身兴趣和实际情况、或者结合大创计划、各类学科竞赛等自主选题。自主选题需经过指导老师审核并批准。选题原则上遵循一人一题、独立完成原则。选题一经确定，不得随意变更，如确需变更，经指导教师同意后，报学院主管领导批准后方可执行。学生应在教学计划规定时间内，在指导教师的指导下，通过自身努力保质保量地完成毕业设计（论文）。

# 计算机与人工智能学院本科毕业设计选题监督制度

为从源头保障毕业设计（论文）的教学质量，特制定如下的毕业设计选题监督机制。

1. 题目应多样化，紧密围绕本专业的培养目标，结合生产、科研和管理等实际问题，强化学生的系统性和综合性训练。
2. 严格控制团队合作完成同一选题数量。
3. 题目需经三级审核，即指导教师和学生确定选题，填写 本科毕业设计（论文）选题在线汇总表； 各系/教研室组建选题质量审核小组，根据专业培养目标和选题要求，审查选题，质量审核小组审核通过后，提交学院审定；学院组织教学指导委员会、教学督导委员会专家、校外企业专家对各专业提交的毕业设计（论文）选题进行审定。
4. 严格控制选题变更率（不超过10%），并做好记录存档。
