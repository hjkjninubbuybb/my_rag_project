# 计算机与人工智能学院普通本科生优秀毕业论文（设计） 推荐规则

根据本科毕业设计（论文）课程的要求和学校年度教学工作安排，学校一般于春季学期末组织开展应届普通本科生优秀毕业论文（设计）评选工作。为方便学生了解并提前准备，现将评选工作重点通知如下，请有意的同学提前准备。

# 一、评选范围

2026 届普通本科毕业生完成的毕业设计（论文）。

# 二、推荐条件

1.符合《郑州大学本科毕业设计（论文）工作规定》要求；
2. “中国知网”论文检测系统查重的文献复制比不得超过15%；
3.推荐参评优秀毕业设计（论文），需经培养单位公示无异议方可报送；
4.推荐参评数量不得超过本培养单位当年普通本科毕业生总数3％；
5.培养单位在限额推荐参评优秀论文以外，如果在中文核心期刊、SCI 来源期刊、CSSCI 来源期刊等期刊上发表（或正式录用）与毕业论文研究内容一致或相近的论文（第一作者署名为学生本人），该毕业论文可以不占本培养单位参评名额，由培养单位统一申报参评优秀毕业论文（需在推荐评优汇总表中备注）。
# 三、推荐流程

1.学生自愿申请或指导老师推荐，学生提交申请表及支撑材料（见四、材料准备）；
2.按最终成绩（百分制成绩）从高到底排名，同等条件下（即最终成绩>90）且三项（指导老师打分、评阅老师打分、答辩委员会打分）均优秀（>90）优先；
3.学院教学督导委员会组织评审并公示评审结果。
# 四、材料准备

1.郑州大学优秀本科毕业论文（设计）推荐表（附件1）、培养单位推荐评优汇总表（需提交word 格式电子文档）（附件2）。
2.参评毕业论文（设计）纸质文档。
3.参评毕业论文（设计）任务书、开题报告、中期检查表及相关答辩纸质材料。
4.每份参评毕业论文（设计）需提交由“中国知网”论文检测系统生成的查重简版报告。
# 五、报送时间及要求

1.每份推优论文（设计）单独装档案袋（含第1、2、3、4 项材料），并打印培养单位名称、专业名称、学生（团队）姓名三项信息粘贴到档案袋封面（凡材料装袋不符合上述要求，一律不予接收）。
2.上述材料需经培养单位主管教学的副院长签字并加盖所在培养单位公章后于指定时间报送至指定地点。
# 六、评审说明

1.教务部对各培养单位推荐的毕业论文（设计）进行资格审查。
2.校优秀毕业论文（设计）评审专家组依据《郑州大学本科毕业论文（设计）工作规定》等要求对各培养单位推荐的毕业论文（设计）进行评审，有关事项另行通知。
# 七、奖励形式

学校为学生颁发校级优秀毕业设计（论文）荣誉证书，为指导老师颁发校级优秀毕业设计（论文）指导老师荣誉证书。

对于获得校级优秀的毕业论文，将在学院设置展示时间和空间，为下一届毕业生提供参考和借鉴。
